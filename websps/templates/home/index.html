Welcome to WebSPS!

WebSPS is a hub for applications used with the Super-Enge Split-Pole Spectrograph at FSU. Currently, as WebSPS is brand new, there is only one application, SPSPlot, which allows you to visualize what states will be present at the focal plane for a given set of reactions and SPS parameters. Hopefully, WebSPS will grow over time and allow everyone to share ideas and tools. Make an account and try out the utilities!

Details

WebSPS is a fullstack python web application using Flask and an SQLite database. As such it is pretty easy to modify and learn (see the Flask documentation for more details on getting started). WebSPS comes with some SPS-related python classes that abstract away most of the physics needed for SPS data and uses pycatima for energy loss calculations. Additionally, WebSPS uses Tailwind-CSS to make everything look pretty. If you want a desktop version of some of the tools here, look for the spspy repository, which has tools like SPSPlot and SPANC.
